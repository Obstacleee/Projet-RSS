Présentation de StreamVoice

I - Contexte

Du 9 au 23 avril 2024, une durée de cinq semaines, nous, les étudiants en troisième année de Bachelor à l'IA School de Toulouse (Johann Cazenave, Lucas Delon, Lucas Sanhes, Mélissa Tchamdja, Dorian Mercier, Joseph Midiburo), avons conçu un site web. Ce site présente, sous format audio, les flux RSS d'informations du journal "Le Monde". Ce projet s'inscrit dans l'application pratique de notre formation en analyse de données.

II - Matériels et Méthodes

    II.1. Données

    Pour réaliser ce projet, nous nous sommes appuyés sur les flux RSS du journal "Le Monde". Vous pouvez consulter ces données à partir des Flux RSS de Le Monde.
    Ces informations sont stockées dans une base de données MySQL.

    II.2. OpenAI API TTS

    Dans la réalisation de ce projet, Nous utiliserons l’API d’OpenAI pour récupérer les données des flux RSS à chaque actualisation grâce à l’intelligence artificielle.
    Pour la conversion des flux rss en audio, nous utiliserons un "text-to-speech" qui est une technologie qui convertit le texte écrit en parole vocale.

    II.3. Modules

    Divers librairies et packages python ont été utilisés dans le cadre de ce projet : scrapper.py, sql.py, TTS2.py, mysql.connector.

    II.3. Site

    Le site web lui a été développé avec de nombreuses technologies de pointe : HTML, CSS, Bootstrap, Flask. Les fichiers audios ont eux été stockés sur Discord. Git a été utilisé pour coordonner le codage du site et faciliter le travail collaboratif.
    NB: Nous vous conseillons d’utiliser Google Chrome comme navigateur pour assurer une meilleure expérience utilisateur.

III - Résultats

    III.2. Notre Application
